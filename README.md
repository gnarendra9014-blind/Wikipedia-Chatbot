# 🤖 Wikipedia Chatbot

A conversational NLP chatbot that fetches real-time information from Wikipedia and answers questions across 15+ topics.

## ✨ Features
- 🔍 Live Wikipedia search for any topic
- 🧠 100+ built-in facts across AI, Science, History, Math, Coding & more
- 💬 Chat history saved automatically
- 📊 Sentiment analysis
- 🧮 Math calculator
- 📝 Summarization
- 🎓 Teach mode — add your own facts

## 🛠️ Tools & Technologies
- Python
- NLTK
- Scikit-learn
- Sentence-Transformers
- Wikipedia API
- NumPy

## ▶️ How to Run

1. Install dependencies:
   pip install nltk scikit-learn numpy sentence-transformers wikipedia-api

2. Run the chatbot:
   python chatbot.py

## 💡 Example Usage
You: tell me about machine learning
Bot: Machine learning enables systems to learn from data...

You: Wikipedia Elon Musk
Bot: [Wikipedia] Elon Musk is a businessman and investor...
